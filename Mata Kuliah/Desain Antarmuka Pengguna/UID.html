IDENTITAS MATA KULIAH

Desain Antarmuka Pengguna
SKS : 3 SKS
Semester : 1
Tahun Akademik : 2024/2025

Deskripsi Mata Kuliah
Mata kuliah dengan kode DIH1A4 - Implementasi Desain Antarmuka Pengguna merupakan matakuliah pada kurikulum 2016 di prodi Diploma 3 Sistem Informasi semester 2. Mata kuliah ini memperlajari Prinsip Dasar User Interface Design, User Profile & User Centered Design. Selain itu pada mata kuliah ini akan mempelajari web desain dengan materi Dasar HTML, Tabel, Form & HTML 5 kemudian memberikan style dengan menggunakan CSS dan diakhir mata kuliah akan diajarkan mengenai konsep dan implementasi dari Web Responsive.

Profil dan Sejarah Mata Kuliah
Mata kuliah Implementasi Desain Antarmuka Pengguna ini memberikan pengetahuan tentang teori dan praktek dalam mendesain antarmuka pengguna berbasis web. Mata kuliah ini juga memberikan mahasiswa kemampuan dan keterampilan dalam menggunakan beberapa tools yang mendukung dalam proses mendesain web dari desain low fidelity sampai high fidelity. Mata kuliah Implementasi Desain Antarmuka Pengguna di program studi ini sudah ada sejak pertama kali Program Studi D3 Manajemen Informatika, Politeknik Telkom pada tahun 2007 yang sekarang sudah berganti nama menjadi D3 Sistem Informasi, Fakultas Ilmu Terapan.
